# 🚪 入口：程序员

> 你有编程经验，可能用过一些AI工具辅助编程。
> 你的优势是技术基础好，学习速度快。

---

## 你的路径
第1步：觉醒（快速过一遍，3天）
→ 01-awakening/ — 重点关注"认知卫生"和"用AI学AI的悖论"

第2步：筑基-数学（1-2周，重点桥梁建设）
→ 你的数学基础够用，重点看每个概念在AI中变成了什么

第3步：筑基-ML/DL/LLM（8-10周）
→ 你的编程能力让你可以边学边实现
→ 推荐资源：Andrej Karpathy的"Zero to Hero"系列

第4步：安全（3-4周）
→ 重点关注代码安全、供应链安全、Prompt注入
→ 参考：编程助手安全指南

第5步：精通（4-6周，这是你的主场）
→ 重点：构建AI解决方案、工作流编排、AI调试和测试
→ 你可以开始创造自己的AI工具了

第6步：超越（持续）
→ 建立持续进化框架

**预估总时长：4-6个月**

→ 现在开始：[01 觉醒](../01-awakening/)（快速过一遍）
  或直接进入 [02 筑基](../02-foundations/)（如果你对觉醒阶段的内容已有认知）
