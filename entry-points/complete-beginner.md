# 🚪 入口：完全零基础

> 你从来没有接触过AI——没关系，这个项目就是为你设计的。

---

## 你现在在哪

你可能：
- 用过ChatGPT但不知道它是怎么工作的
- 听说过AI但觉得它离自己很远
- 对AI好奇但不知道从哪里开始
- 担心AI会影响你的工作

## 你的路径
第1步：觉醒（1周）
→ 从这里开始：01-awakening/
→ 理解AI的本质，建立正确的认知基础

第2步：筑基-数学桥梁（2-3周）
→ 02-foundations/mathematics/
→ ⚠️ 需要大学数学基础。如果没有，先看推荐的视频课程补齐。

第3步：筑基-从ML到LLM（12-16周）
→ 按顺序学习机器学习→深度学习→大模型内核→基础设施→数据→评估
→ 这是最长也最重要的阶段

第4步：安全（4周）
→ 03-security/
→ 学会安全地使用和部署AI

第5步：精通和超越（8-10周）
→ 04-mastery/ → 05-transcendence/
→ 从使用者变成创造者，建立持续进化框架


**预估总时长：6-9个月（每周10-15小时）**
