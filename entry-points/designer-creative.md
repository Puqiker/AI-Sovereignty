# 🚪 入口：设计师/创意工作者

> 你关注的是AI和创造力的关系——
> AI如何帮助你的创作，以及如何保持你的创造力不被AI侵蚀。

---

## 你的路径
第1步：觉醒（1周）— 必修
→ 特别关注"人类心智 vs AI"——理解你的创造力为什么是不可替代的
→ "认知卫生"对创意工作者尤其重要

第2步：筑基（选择性，2-3周）
→ 重点理解：大模型是怎么生成内容的（概率采样）
→ 理解图像生成模型的基本原理
→ 不需要深入数学和编程

第3步：安全（2周）
→ 重点：图像生成工具安全、版权问题、深度伪造
→ 数据隐私（你的创作数据去了哪里）

第4步：精通（选择性，2周）
→ 重点：场景发现方法论（在你的创作流程中找到AI介入点）
→ 工具设计哲学（如果你想创造面向创意工作者的AI工具）

第5步：超越（推荐，1-2周）
→ 核心重点：创造力放大、潜意识与直觉
→ 这些内容直接关系到你作为创意工作者的核心竞争力


**预估总时长：2-3个月**

→ 现在开始：[01 觉醒](../01-awakening/)
