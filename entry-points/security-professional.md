# 🚪 入口：安全从业者

> 你有传统安全（网络安全、应用安全）的背景，
> 想要进入AI安全领域。

---

## 你的路径
第1步：觉醒（快速，3天）
→ 快速建立对AI的基本认知

第2步：筑基（重点选择，4-6周）
→ 必须理解：Transformer、预训练、微调、RLHF、推理机制
→ 这些是你理解AI安全威胁根因的前提
→ 数学基础看你的背景——如果有，可以快速过

第3步：安全（核心重点，6-8周）
→ 全部深入学习
→ 你的传统安全经验在这里有巨大优势：
- Prompt注入 ↔ SQL注入（同类问题）
- 供应链攻击 ↔ 软件供应链安全（同类框架）
- 红队方法论 ↔ 渗透测试（同类方法）
→ 重点深入：安全体系建设、自适应安全系统

第4步：精通（选择性，2-3周）
→ 重点：AI调试、AI测试（从安全角度的测试设计）
→ Agent安全、工作流威胁

第5步：超越（推荐，1周）
→ 快速过


**预估总时长：3-5个月**

  然后直接进入 [02 筑基](../02-foundations/)
