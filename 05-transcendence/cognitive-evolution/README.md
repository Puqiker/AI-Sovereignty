# 认知进化

> AI在进步，你也需要进步。
> 但你的进步方向不应该是"变得更像AI"——
> 那是一条你永远赢不了的路。
> 你的进步方向应该是"变得更像人应该成为的样子"——
> 更有创造力、更有判断力、更有智慧。
> 这些恰恰是AI做不到的。
