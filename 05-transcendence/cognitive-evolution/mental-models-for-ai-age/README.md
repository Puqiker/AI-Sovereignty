# AI时代的心智模型

> 心智模型是你理解世界的内在框架。
> 框架对了，你看什么都能看到本质。
> 框架错了，你看到的都是表象。
> AI时代需要一些新的心智模型。

---

## 模型一：工具放大论

**AI是放大器，不是替代器。**

锤子放大了你手臂的力量，但不替代你的手臂。
望远镜放大了你眼睛的视力，但不替代你的眼睛。
AI放大了你的认知能力，但不替代你的认知。

这个模型的推论：
- AI的价值取决于被放大的东西的质量——如果你的思考是浅薄的，AI放大的也是浅薄
- 提升自己的思考质量比提升AI使用技能更重要
- AI让强者更强——因为被放大的基础更好

### 怎么用这个模型

当你准备用AI做某件事时，问自己：
"如果没有AI，我自己能做到什么程度？"

如果答案是"完全不能"——那AI不是在放大你，而是在替代你。
这种情况下你需要先建立自己的基础能力。

如果答案是"能做但比较慢/效果一般"——那AI是在放大你。
这是健康的人机协作模式。

## 模型二：认知层级论
高层：方向判断 / 价值选择 / 问题定义 / 创新突破
→ 这些是人应该做的，AI目前做不好

中层：方案设计 / 分析推理 / 知识应用
→ 人和AI协作，人主导判断，AI辅助执行

低层：信息搜集 / 格式转换 / 重复性处理
→ 这些可以大量交给AI

你的目标是**持续上移**——把越来越多的低层工作交给AI，
让自己有更多的时间和精力投入到高层的认知活动中。

如果你发现自己一直在用AI做低层工作，
但从来不用省下来的时间做高层的思考——
那你没有在进化，你只是在偷懒。

## 模型三：能力半衰期论

在AI时代，不同类型的能力有不同的"半衰期"——
有些能力很快贬值，有些能力越来越值钱。

快速贬值的能力（半衰期短）：
- 特定工具的操作技能
- 记忆型知识（事实、数据、流程）
- 模板化的执行能力

缓慢贬值甚至升值的能力（半衰期长）：
- 批判性思维和判断力
- 跨领域的连接能力
- 提出好问题的能力
- 理解原理的能力
- 创造力和想象力
- 人际沟通和领导力


**你应该把有限的学习时间投资在半衰期长的能力上。**

这就是为什么这个项目强调"理解原理"而非"学会操作"——
原理的半衰期远长于任何具体工具的操作方法。

## 模型四：信息过滤器论

AI时代的信息量呈指数增长。
每天有新工具、新论文、新技术、新观点。

你不可能跟踪所有信息。你需要一个过滤器。

**你的过滤器应该基于原理和本质，而不是表面的热度。**

当一个新的AI概念出现时：
- 它的底层原理是否是你已经理解的东西的变种？→ 快速评估，不必深入
- 它是否在原理层面有根本性的创新？→ 值得深入研究
- 它解决的问题和你的需求相关吗？→ 决定是否投入时间

这个过滤器让你在信息洪流中保持清醒，
而不是被每一个新事物牵着走。

---

## 关键认知

1. AI是放大器——提升被放大的基础比提升放大器的使用技能更重要
2. 持续向认知层级的上层移动——低层交给AI，高层留给自己
3. 投资半衰期长的能力——原理、判断力、创造力
4. 建立基于原理的信息过滤器——不被信息洪流淹没
