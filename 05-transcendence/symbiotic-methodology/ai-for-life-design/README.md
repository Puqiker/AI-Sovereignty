# AI与生活设计

---

## AI不只是工作工具

AI可以帮助你更好地管理和设计你的生活——
不是让AI替你做决定，而是用AI辅助你的自我管理。

### 学习加速

- 用AI来解释你不理解的概念（但记住用AI学AI的悖论——要验证）
- 用AI帮你制定学习计划和路径
- 用AI来做练习和测验——生成适合你水平的题目

### 决策支持

- 面对复杂决策时，用AI帮你整理利弊
- 让AI扮演"魔鬼代言人"——故意反驳你的观点，帮你发现盲点
- 但最终决策是你的——AI是顾问，不是决策者

### 健康管理

- 用AI辅助理解健康信息（但重要的健康决策必须咨询医生）
- 用AI帮你设计运动计划、饮食方案
- 注意：AI不是医生，健康建议需要专业验证

### 边界意识

AI可以辅助你的生活，但它不应该成为你生活的中心。

如果你发现自己每一个小决定都要先问AI——
穿什么衣服、吃什么饭、怎么回复一条消息——
那你不是在用AI提升生活，你是在让AI接管生活。

**保持对生活的直接参与感和掌控感。**

---

## 关键认知

1. AI是生活的辅助工具——不是生活的指挥者
2. 学习、决策、健康管理都可以获得AI的辅助
3. 最终决策权始终在你手中
4. 保持对生活的直接参与感——不要让AI替你生活
