# 自我评估系统

---

## 定期自我评估

每个季度花一个小时，用以下框架评估自己的状态：

### 认知维度
AI理解深度：
□ 我能否解释当前最新的AI技术的原理？
□ 我能否判断一个新出现的AI工具的本质和局限？
□ 面对AI相关的争议，我有自己的独立判断吗？

安全意识：
□ 我知道我使用的AI工具的数据处理策略吗？
□ 我最近有没有审查过我的AI使用中的安全风险？
□ 我能否识别新出现的AI安全威胁？

独立思考：
□ 我最近有没有在没有AI帮助的情况下独立解决一个复杂问题？
□ 我在面对AI的输出时，是否保持了批判性思维？
□ 我是否过度依赖AI做决策？

### 能力维度
创造能力：
□ 我最近有没有创造出某个新的东西（产品、内容、方案）？
□ 这个创造中有多少是我的独特贡献，多少是AI的？
□ 我的创造力是在提升还是在退化？

构建能力：
□ 我能否独立设计和构建一个AI解决方案？
□ 我的技术能力是否跟上了AI技术的发展？
□ 我最近学了什么新的技术知识？

影响力：
□ 我的AI能力是否在帮助我的职业/事业发展？
□ 我是否在帮助身边的人理解和使用AI？
□ 我的AI能力是否转化为了经济价值？

### 生活维度

平衡感：
□ AI是否在帮我节省时间，而我把时间投入了有意义的事情？
□ 我是否保持了足够的线下体验和人际交流？
□ 我是否对自己的生活有掌控感？


---

## 评估产出

每次评估后，明确：
1. 我在哪些方面做得好——继续保持
2. 我在哪些方面有退步——需要立即关注
3. 我下一个季度的重点提升方向是什么

---

## 关键认知

1. 定期自我评估是保持进化的前提——你必须知道自己在哪里
2. 评估覆盖认知、能力、生活三个维度——不只是技术能力
3. 每次评估产出具体的行动方向——不是泛泛地"我要更努力"
