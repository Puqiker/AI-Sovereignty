# 进度追踪

---

## 长期进化轨迹

保持一份个人的"认知进化日志"——
不需要每天写，但至少每个月记录一次：

日期：2026-03-XX

本月的关键学习/成长：
- 学习了什么新知识
- 完成了什么项目/作品
- 认知上有什么新的领悟

本月的AI使用反思：
- AI在哪些方面帮助了我
- 我是否过度依赖AI的某些方面
- 我的独立能力是否保持/提升

下月的重点：
- 要深入的方向
- 要改进的习惯
- 要完成的目标


---

## 为什么追踪很重要

没有追踪，你不知道自己是在进化还是在原地踏步。

人的认知变化是缓慢的——你可能感觉自己一直在学习，
但实际上只是在同一个层面上重复。

定期回顾你的记录，看三个月前的你和现在的你——
思维方式有变化吗？能力有提升吗？认知有深化吗？

如果答案是"没什么变化"——你需要调整你的学习和成长策略。
