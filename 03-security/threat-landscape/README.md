# 威胁全景

> 这个模块覆盖AI系统面临的所有主要威胁类型。
> 每种威胁都从"为什么能成功"讲起——追溯到你在筑基阶段学过的原理。
> 理解了根因，你就不需要背诵攻击方式的清单——
> 你能自己推导出新的攻击方式，也能设计出更根本的防御。

## 学习顺序

| 顺序 | 威胁类型 | 为什么重要 |
|------|---------|-----------|
| 1 | [Prompt注入](prompt-injection/) | 最常见、最容易实施的攻击方式 |
| 2 | [数据投毒](data-poisoning/) | 从源头污染模型能力的攻击 |
| 3 | [模型窃取](model-extraction/) | 商业和知识产权层面的威胁 |
| 4 | [对抗性攻击](adversarial-attacks/) | 通过精心设计的输入欺骗模型 |
| 5 | [隐私泄露](privacy-leakage/) | 模型可能泄露训练数据中的敏感信息 |
| 6 | [供应链攻击](supply-chain-attacks/) | 通过第三方组件渗透AI系统 |
| 7 | [新兴威胁](emerging-threats/) | 正在出现的、尚未被充分研究的威胁 |
| 8 | [AI工作流威胁](ai-workflow-threats/) | 多模型组合系统的特有安全问题 |
