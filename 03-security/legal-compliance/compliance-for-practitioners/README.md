# 从业者合规指南

---

## 你需要关注的合规维度

### 1. 数据隐私

- 收集和使用个人数据需要法律依据（GDPR的合法基础、中国的个人信息保护法）
- 用户有权知道他们的数据是否被用于AI训练
- 数据跨境传输可能需要额外的法律保障

### 2. AI决策的透明度和可解释性

- 在某些领域（贷款审批、招聘筛选），AI决策可能需要可解释
- 用户有权知道AI是否参与了影响他们的决策
- 自动化决策可能需要提供人工复审的选项

### 3. 知识产权

- AI生成内容的版权归属在法律上尚不完全明确
- 使用受版权保护的材料训练AI可能面临法律风险
- 在商业场景中使用AI生成内容前，了解相关的IP风险

### 4. 行业特定合规
金融行业：模型风险管理（SR 11-7）、公平借贷法
医疗行业：HIPAA数据保护、医疗器械法规（AI作为SaMD）
教育行业：学生数据保护（FERPA）、公平性要求

### 5. 内容责任

- 你对AI生成并发布的内容承担责任
- AI生成的误导性或有害内容可能导致法律后果
- 建立AI生成内容的审核机制
