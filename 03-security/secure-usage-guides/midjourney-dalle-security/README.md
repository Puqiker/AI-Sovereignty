# 图像生成工具安全指南

> 涵盖Midjourney、DALL-E、Stable Diffusion等图像生成工具。

---

## 核心安全关注点

### 1. 版权与知识产权

- AI生成的图像的版权归属在法律上尚不明确
- 训练数据可能包含受版权保护的作品——生成的图像可能在风格上"类似"特定艺术家
- 商业使用时需要审查许可条款

### 2. 深度伪造风险

- 不要使用这些工具生成真实人物的虚假图像
- 注意工具对人脸生成的限制策略
- 了解你所在地区关于深度伪造的法律规定

### 3. 数据安全

- 上传的参考图片会被传输到服务商服务器
- Midjourney在Discord中使用——注意Discord频道的可见性设置
- Stable Diffusion可以本地运行——对于敏感图像处理，优先考虑本地部署

### 4. 内容安全

- 大多数工具有内容过滤策略——但过滤不是完美的
- 生成的图像应在发布前进行人工审查
- 企业使用时需要建立内容审核流程

## 使用红线

- ❌ 不要生成真实人物的虚假/误导性图像
- ❌ 不要上传包含敏感信息的图片作为参考
- ⚠️ 商业使用前确认许可条款
- ⚠️ 生成的图像在发布前应经过审核
