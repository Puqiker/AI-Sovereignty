## 通用安全原则

在进入具体工具之前，以下原则适用于所有AI工具：

### 1. 了解你的数据去了哪里

每个AI工具背后都有一条数据流：
你的输入 → [传输] → [服务商服务器] → [模型处理] → [输出返回]
↓
是否被存储？
是否被用于训练？
是否有人类审核？
存储在哪个地区？

**使用任何AI工具之前，先弄清楚这条数据流的每一个环节。**

### 2. 数据分级
公开数据：可以在任何AI工具中使用
内部数据：只在企业版/私有部署的AI工具中使用
机密数据：只在完全私有部署的环境中使用
绝密数据：不应该输入任何AI工具

### 3. 最小化输入

只输入完成任务所需的最少信息。
不要把整个数据库导出粘贴进去——提取相关部分即可。

### 4. 输出验证

AI的输出不是最终结果——它是需要被审查的草稿。
特别是涉及事实、数字、代码安全的输出，必须人工验证。
