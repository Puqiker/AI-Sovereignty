# ChatGPT 安全使用指南

---

## 数据流分析
你的输入 → HTTPS加密传输 → OpenAI服务器（美国）→ 模型处理 → 输出返回
↓
免费版/Plus版：默认可能用于训练
Team/Enterprise版：不用于训练
API：不用于训练（默认）


## 关键安全配置

### 1. 关闭训练数据共享（免费版/Plus版）

Settings → Data Controls → "Improve the model for everyone" → 关闭

关闭后，你的对话内容将不会被用于训练模型。
但注意：OpenAI仍可能存储你的对话用于安全监控和滥用检测。

### 2. 临时对话模式

使用"Temporary Chat"模式——对话不会被保存到历史记录中。
适合处理敏感话题时使用。

### 3. 自定义GPT的安全

如果你使用自定义GPT（GPTs）：
- 审查GPT的指令——有些第三方GPT可能包含不安全的指令
- 注意GPT请求的权限——是否需要访问外部API
- 自定义GPT的创建者可能能看到使用情况

## 使用红线

- ❌ 不要输入密码、API密钥、Token等凭证
- ❌ 不要输入完整的个人身份信息（身份证号、社保号等）
- ❌ 不要输入公司核心商业机密（未公开的财务数据、核心算法等）
- ❌ 不要在免费版中进行涉及敏感数据的对话
- ⚠️ 生成的代码必须经过安全审查才能用于生产环境
- ⚠️ 事实性信息必须交叉验证

## 企业环境

- 使用ChatGPT Enterprise或API——确保数据不用于训练
- 制定公司级的ChatGPT使用政策
- 要求员工了解数据分级规则
- 考虑是否需要API调用审计日志
