# Claude 安全使用指南

---

## 数据流分析

你的输入 → HTTPS加密传输 → Anthropic服务器 → 模型处理 → 输出返回
↓
免费版/Pro版：默认不用于训练
API：不用于训练
可能被存储用于安全审计（有保留期限）


## 关键安全配置

### 数据处理策略

Anthropic的默认策略是不使用用户对话来训练模型。
但建议确认最新的隐私政策，因为策略可能更新。

API使用时的数据保留：
- API输入/输出默认保留30天用于信任与安全审查
- 可以通过申请零保留策略来缩短保留期

### 项目与对话管理

- 利用Projects功能组织不同用途的对话，避免跨场景的上下文混合
- 敏感对话使用独立的Project

## 使用红线

- ❌ 同ChatGPT：不输入凭证、完整个人信息、核心商业机密
- ⚠️ 长文档分析时注意：整个文档内容都会被传输到Anthropic服务器
- ⚠️ 使用Claude处理的文件内容需要符合你的数据分级策略

## 企业环境

- API使用时配置适当的数据保留策略
- 评估是否需要与Anthropic签订数据处理协议（DPA）
- 考虑通过AWS Bedrock等渠道使用Claude以获得更多的数据控制
