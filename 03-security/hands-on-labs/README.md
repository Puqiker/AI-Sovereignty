# 实战安全实验室

> 理论知识必须通过实践来巩固。
> 这个模块提供受控环境中的攻防实践，
> 让你亲手体验攻击和防御的过程。

---

## 实验室类型

| 实验室 | 内容 |
|--------|------|
| [攻击模拟](attack-simulation/) | 在安全环境中模拟各种AI攻击 |
| [防御演练](defense-exercises/) | 实践各种防御措施的部署和配置 |
| [渗透测试](penetration-testing/) | 对AI系统进行完整的安全评估 |

## 重要说明

⚠️ **所有实验必须在受控环境中进行。**

- 只在你有权测试的系统上进行安全测试
- 不要对第三方系统进行未经授权的渗透测试
- 遵循负责任披露原则——如果你发现了真实系统的漏洞，通过官方渠道报告
