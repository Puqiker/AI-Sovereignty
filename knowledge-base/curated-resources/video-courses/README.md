## 觉醒阶段

### ⭐ 3Blue1Brown - But what is a neural network?
- **语言**：英语（有中文字幕）
- **难度**：入门
- **时长**：约1小时（系列4集）
- **关键词**：神经网络直觉、可视化、梯度下降、反向传播
- **推荐理由**：用动画可视化解释神经网络，是建立直觉的最佳入门资源。不需要任何前置知识。
- **链接**：https://www.youtube.com/playlist?list=PLZHQObOWTQDNU6R1_67000Dx_ZCJB-3pi
- **对应模块**：[AI到底是什么](../../../01-awakening/what-is-ai-really/)

## 数学基础

### ⭐ 3Blue1Brown - Essence of Linear Algebra
- **语言**：英语（有中文字幕）
- **难度**：入门-进阶
- **时长**：约4小时（系列16集）
- **关键词**：线性代数、向量、矩阵变换、特征值、直觉
- **推荐理由**：最好的线性代数直觉建立资源。用几何动画把抽象的线性代数概念变得可视化。不是教你做题，而是让你真正理解这些概念在做什么。
- **链接**：https://www.youtube.com/playlist?list=PLZHQObOWTQDPD3MizzM2xVFitgF8hE_ab
- **对应模块**：[线性代数 → AI](../../../02-foundations/mathematics/linear-algebra/)

### ⭐ 3Blue1Brown - Essence of Calculus
- **语言**：英语（有中文字幕）
- **难度**：入门-进阶
- **时长**：约3小时（系列12集）
- **关键词**：微积分、导数、积分、直觉、可视化
- **推荐理由**：如果你学过微积分但忘了或者从没真正理解过，这个系列帮你重建直觉。
- **链接**：https://www.youtube.com/playlist?list=PLZHQObOWTQDMsr9K-rj53DwVRMYO3t5Yr
- **对应模块**：[微积分与优化 → AI](../../../02-foundations/mathematics/calculus-optimization/)

### StatQuest - 统计学基础系列
- **语言**：英语
- **难度**：入门
- **关键词**：概率、统计、贝叶斯、分布、假设检验
- **推荐理由**：Josh Starmer用极其通俗的方式讲解统计概念，每集5-15分钟，适合碎片化学习。
- **链接**：https://www.youtube.com/c/joshstarmer
- **对应模块**：[概率统计 → AI](../../../02-foundations/mathematics/probability-statistics/)
