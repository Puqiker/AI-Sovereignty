# 推荐书籍

---

## 觉醒阶段

### 《AI 3.0》/ Melanie Mitchell
- **关键词**：AI历史、AI能力与局限、通用智能
- **推荐理由**：从科学角度审视AI的真实能力和局限，帮助你建立不被营销炒作左右的认知。
- **语言**：英文（有中文译本）

---

## 数学基础

### 《线性代数应该这样学》(Linear Algebra Done Right) / Sheldon Axler
- **关键词**：线性代数、抽象思维、向量空间
- **推荐理由**：注重概念理解而非计算技巧，和本项目"理解原理"的理念一致。
- **语言**：英文（有中文译本）

### 《概率论与数理统计》/ 陈希孺
- **关键词**：概率论、数理统计、中文教材
- **推荐理由**：中文概率统计教材中的经典，讲解清晰严谨。
- **语言**：中文

---

## 机器学习

### ⭐《统计学习方法》/ 李航
- **关键词**：机器学习、统计学习、算法推导
- **推荐理由**：中文机器学习领域的经典教材，数学推导严谨，覆盖核心ML算法。
- **语言**：中文

### 《Pattern Recognition and Machine Learning》/ Christopher Bishop
- **关键词**：模式识别、贝叶斯方法、机器学习理论
- **推荐理由**：从贝叶斯视角系统性讲解机器学习。理论深度高，适合想深入理解原理的读者。
- **语言**：英文

---

## 深度学习

### ⭐《深度学习》(Deep Learning) / Ian Goodfellow, Yoshua Bengio, Aaron Courville
- **关键词**：深度学习教科书、神经网络、优化、正则化
- **推荐理由**：被称为"花书"，深度学习领域的权威教科书。覆盖了从基础到前沿的完整内容。
- **语言**：英文（有中文译本）
- **免费在线版**：https://www.deeplearningbook.org/

### 《动手学深度学习》(Dive into Deep Learning) / 李沐等
- **关键词**：深度学习实践、PyTorch、代码实现
- **推荐理由**：理论与代码并行，每个概念都有可运行的代码实现。中文内容质量极高。
- **语言**：中英双语
- **免费在线版**：https://d2l.ai/

---

## 大模型

### 《Build a Large Language Model (From Scratch)》/ Sebastian Raschka
- **关键词**：LLM实现、从零构建、GPT架构
- **推荐理由**：从零实现一个完整的大语言模型，适合想要亲手理解LLM每一个组件的读者。
- **语言**：英文

---

## 工程实战

### 《Designing Machine Learning Systems》/ Chip Huyen
- **关键词**：ML系统设计、生产环境、数据工程、监控
- **推荐理由**：从工程角度讲解ML系统的完整生命周期，填补了学术课程和生产实践之间的鸿沟。
- **语言**：英文
