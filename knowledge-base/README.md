# 📚 知识库

> 这个知识库是整个项目的资源中枢。
> 它为每个阶段的学习提供经过筛选的外部资源，
> 包括视频课程、经典论文、推荐书籍、播客和交互式教程。
>
> **选择标准**：
> 1. 内容质量经过社区广泛验证
> 2. 讲解深度符合本项目"原理优先"的理念
> 3. 优先推荐免费资源，付费资源标注价格
> 4. 中英文资源兼顾
>
> **时效性说明**：
> 外部链接可能随时间失效。每个资源条目保留了关键词元数据，
> 即使链接失效，你仍然可以用关键词搜索找到相关内容。
> 配套的 [链接健康检查工具](../tools/link-health-checker/) 会定期检查链接有效性。
