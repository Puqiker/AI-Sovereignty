# 深度学习

> 机器学习告诉你"模型从数据中学习"。
> 深度学习告诉你：**当模型足够深、数据足够多时，会发生什么。**
> 答案是：涌现出令人惊讶的能力。
>
> 这个模块从最基本的神经网络开始，
> 带你一路走到Transformer——当前几乎所有主流AI模型的底层架构。
> 到这里结束时，你将理解大语言模型的物理结构。

---

## 什么是"深度"

"深度学习"中的"深度"指的是神经网络的层数。

一个只有一层的网络叫浅层网络。
一个有几十层、几百层甚至上千层的网络叫深度网络。

为什么"深"很重要？

你在数学基础的线性代数模块学过：
每一层做的是线性变换+非线性激活。
单独一层能学到的模式是有限的。
但当你把几十上百层叠起来时，
网络能学到极其复杂的、层层递进的特征表示。

一个直觉性的例子——图像识别：
- 第一层：学会识别边缘（横线、竖线、对角线）
- 中间层：把边缘组合成纹理和局部形状（眼睛、鼻子、耳朵的轮廓）
- 更高层：把局部形状组合成完整的对象（一张猫脸）
- 最后层：把对象和类别关联起来（"这是猫"）

**每一层在前一层的基础上构建更高级的抽象。**
这种从简单到复杂的层级表示学习，是深度学习的核心优势。

浅层网络理论上也能学习任意复杂的函数，
但它需要指数级宽度的隐藏层，实际上不可行。
深度网络通过增加层数来高效地表示复杂函数。

## 学习顺序

这五个模块按照技术的演进顺序排列。
每一个架构的出现都是为了解决前一个架构的局限性：

| 顺序 | 模块 | 核心内容 |
|------|------|---------|
| 1 | [神经网络基础](neural-network-fundamentals/) | 一切的起点：神经元、层、激活函数、前向/反向传播 |
| 2 | [CNN架构](cnn-architectures/) | 为什么图像识别不用全连接：卷积的思想 |
| 3 | [RNN与序列模型](rnn-lstm-sequence/) | 处理有顺序的数据：文本、时间序列 |
| 4 | [注意力机制](attention-mechanism/) | 解决RNN的根本缺陷：让模型学会"关注"重要信息 |
| 5 | [Transformer架构](transformer-architecture/) | 当前AI的统一架构：注意力就是你所需要的一切 |

你会看到一条清晰的演进线索：
全连接网络 → 太多参数，无法处理结构化数据
↓
CNN → 利用空间局部性，大幅减少参数，图像处理的突破
↓
RNN → 处理序列数据，但记忆有限，无法并行
↓
注意力机制 → 让模型直接关注远距离的相关信息
↓
Transformer → 完全基于注意力，抛弃递归，可大规模并行
↓
当前所有主流大模型的基础


每一步都是在解决前一步的瓶颈。理解这条演进线索，
比死记每个架构的细节更重要。

## 完成标准

当你完成深度学习模块后，你应该能够：

- [ ] 画出一个简单神经网络的结构，解释数据如何从输入流向输出
- [ ] 解释为什么CNN对图像有效而全连接网络不行
- [ ] 解释RNN的"记忆"为什么是有限的，以及LSTM如何缓解这个问题
- [ ] 用自己的话解释注意力机制在做什么——不是背公式，而是说出直觉
- [ ] 画出Transformer的核心结构，解释自注意力、多头注意力、位置编码各自的作用
- [ ] 理解为什么Transformer取代了RNN成为主流——并行能力和长距离依赖处理

完成后，继续进入 [大模型内核](../llm-internals/)。
