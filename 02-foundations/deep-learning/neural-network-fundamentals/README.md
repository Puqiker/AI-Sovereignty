# 神经网络基础

> 这是一切深度学习的起点。
> 你在数学基础模块已经知道了神经网络的每一层在数学上做什么：矩阵乘法+激活函数。
> 这个模块把这些数学组件组装成一个完整的系统，
> 让你看到一个神经网络从输入到输出的全貌。

---

## 神经元：最小的计算单元

一个人工神经元做的事情非常简单：
输出 = 激活函数(加权求和 + 偏置)
= f(w₁x₁ + w₂x₂ + ... + wₙxₙ + b)
= f(w · x + b)

它接收多个输入，每个输入乘以一个权重，
全部加起来再加一个偏置，
然后通过激活函数产生输出。

这和你学过的线性回归几乎一样——
唯一的区别是最后多了一个激活函数。

单个神经元只能学习线性的分界面（在二维中就是一条直线）。
但当你把大量神经元组织成层，再把多层叠起来时，
网络就能学习任意复杂的非线性关系。

---

## 层：神经元的组织方式

### 全连接层（Dense Layer / Fully Connected Layer）

最基本的层类型：前一层的每个神经元和后一层的每个神经元之间都有连接。
输入层 隐藏层 输出层
x₁ ——→ ○ ○ ○ ——→ ○ ○
x₂ ——→ ○ ○ ○ ——→ ○ ○
x₃ ——→ ○ ○ ○

在数学上，一个全连接层就是：
y = f(Wx + b)


- W 是权重矩阵，维度为 (输出神经元数 × 输入神经元数)
- x 是输入向量
- b 是偏置向量
- f 是激活函数

**你在数学基础已经知道这些了。** 现在的关键是理解多层叠加的效果。

### 为什么需要多层

假设你要学习一个复杂的分类边界——比如在二维平面上把数据分成两类，
但分界线不是一条直线，而是一条弯曲的曲线。

一层网络只能画一条直线。
两层网络可以画折线。
多层网络可以画任意形状的曲线。

层数越深，网络能表达的函数越复杂。

数学上有一个定理叫**通用逼近定理（Universal Approximation Theorem）**：
一个有足够宽的隐藏层的神经网络可以逼近任意连续函数。

但"足够宽"在实际中往往意味着不切实际地宽。
增加深度比增加宽度更高效——这就是"深度"学习优于浅层学习的数学原因。

## 激活函数：为什么非线性是必需的

### 没有激活函数会怎样

如果去掉激活函数，一个三层网络的计算是：
y = W₃(W₂(W₁x + b₁) + b₂) + b₃
= W₃W₂W₁x + (W₃W₂b₁ + W₃b₂ + b₃)
= W'x + b'

三个线性变换的组合仍然是一个线性变换。
不管你叠多少层，效果和一层一样。

**没有非线性，深度就没有意义。**

激活函数引入非线性，使得每一层在线性变换之后"扭曲"空间，
这样多层叠加才能产生比单层更强的表达能力。

### 主要的激活函数

**Sigmoid**：

σ(x) = 1 / (1 + e^(-x))


输出范围 (0, 1)。
早期神经网络的标准选择。
问题：当输入很大或很小时，导数接近零（梯度饱和），导致梯度消失。

**Tanh**：
tanh(x) = (e^x - e^(-x)) / (e^x + e^(-x))

输出范围 (-1, 1)。比Sigmoid好一些（零中心化），但仍有梯度饱和问题。

**ReLU（Rectified Linear Unit）**：
ReLU(x) = max(0, x)

x大于0时输出x，x小于0时输出0。
极其简单，计算快，且正区间导数恒为1——**不存在梯度饱和问题**。

这就是你在数学基础微积分模块看到的：
链式法则中如果每一层的导数都是1（正区间的ReLU），
乘积链不会衰减，梯度能顺利传回前面的层。

ReLU是当前深度学习中最常用的激活函数，正是因为它解决了梯度消失问题。

**GELU（Gaussian Error Linear Unit）**：
GELU(x) = x × Φ(x) （Φ是标准正态分布的累积分布函数）

可以理解为"平滑版的ReLU"——在零附近不是硬切换，而是平滑过渡。
Transformer架构中广泛使用。

你不需要记住每个激活函数的公式。
你需要理解的核心是：**激活函数的作用是引入非线性，没有它深度就没有意义。**

## 前向传播：数据如何流过网络

前向传播就是数据从输入层到输出层的完整计算过程。

以一个三层网络为例：
输入 x
↓
第一层：h₁ = f₁(W₁x + b₁)
↓
第二层：h₂ = f₂(W₂h₁ + b₂)
↓
第三层（输出层）：ŷ = softmax(W₃h₂ + b₃)
↓
输出：每个类别的概率分布

每一步都是你在数学基础已经熟悉的操作：
矩阵乘法（线性变换）→ 加偏置 → 激活函数（非线性）。

前向传播的过程是确定性的——给定相同的输入和相同的参数，输出完全一样。

---

## 反向传播：网络如何学习

你在数学基础微积分模块已经学过了反向传播的数学本质——链式法则。

现在把它放到具体的网络结构中来看：
前向传播（计算预测结果和损失）：
x → h₁ → h₂ → ŷ → Loss

反向传播（计算梯度）：
∂L/∂W₃ ← ∂L/∂ŷ ← Loss
∂L/∂W₂ ← ∂L/∂h₂ ← ∂L/∂ŷ
∂L/∂W₁ ← ∂L/∂h₁ ← ∂L/∂h₂


梯度从损失开始，沿着计算路径反向流回每一层，
每经过一层就乘以该层的局部梯度。

然后用这些梯度做梯度下降，更新每一层的参数。

**前向传播计算预测，反向传播计算梯度，梯度下降更新参数。**
这三个步骤构成了训练的一次完整迭代。

### 计算图的视角

现代深度学习框架（PyTorch、TensorFlow）使用**计算图**来自动实现反向传播。

前向传播时，框架在背后记录每一步运算形成一张图。
调用反向传播时，框架沿着图反向遍历，自动计算每个参数的梯度。

你不需要手动推导每一层的导数——框架帮你做了。
但你需要理解它在做什么，这样当出问题时你能诊断原因。

## 训练的实际过程

把所有东西串起来，训练一个神经网络的实际流程：
初始化
随机初始化所有权重矩阵和偏置

循环以下步骤（每一轮叫一个epoch）：

a. 把训练数据分成小批次（mini-batch），每批比如32或64个样本

b. 对每个mini-batch：

前向传播：计算预测值
计算损失：预测值和真实标签的差距
反向传播：计算每个参数的梯度
参数更新：θ = θ - η × ∇L（梯度下降）
c. 一个epoch结束后（所有训练数据都过了一遍），
在验证集上评估性能

d. 如果验证集性能不再提升，停止训练（早停策略）

最终在测试集上评估模型性能


### 关键的超参数

训练过程中有一些需要你设定而不是模型自己学习的参数，叫**超参数**：

- **学习率（Learning Rate）**：每一步走多大。太大会震荡甚至发散，太小训练太慢。这是最重要的超参数。
- **批次大小（Batch Size）**：每次用多少样本来估计梯度。大批次梯度估计更准但需要更多内存，小批次噪声更大但能探索更多。
- **层数和宽度**：网络有多深、每层有多少神经元。越大的网络能表达越复杂的函数，但也更容易过拟合、需要更多数据和算力。
- **训练轮数（Epochs）**：数据过多少遍。太少欠拟合，太多过拟合。

## 正则化：防止过拟合

大型神经网络参数众多，很容易过拟合——在训练数据上表现完美但在新数据上表现差。

几种主要的正则化技术：

### Dropout

训练时随机"关闭"一定比例的神经元（比如50%）。

训练时：h = f(Wx + b) × mask （mask中50%的位置为0）
推理时：h = f(Wx + b) × 0.5 （用所有神经元但缩放输出）


效果：迫使网络不依赖任何单个神经元，而是学习冗余的表示。
这减少了过拟合，提高了泛化能力。

直觉上可以理解为：每次训练都在用一个略有不同的子网络，
最终模型是很多子网络的平均效果——类似于集成学习。

### 权重衰减（Weight Decay / L2正则化）

在损失函数中加一个惩罚项，惩罚过大的权重值：
总损失 = 原始损失 + λ × Σ|wᵢ|²


效果：迫使权重保持较小的值，防止模型对训练数据中的噪声做出过度反应。

### 批归一化（Batch Normalization）

在每一层的输出上做归一化——使其均值为0、方差为1：
x̂ = (x - μ_batch) / √(σ²_batch + ε)
y = γx̂ + β （γ和β是可学习的参数）


效果：稳定训练过程，允许使用更大的学习率，加速收敛。
它也有一定的正则化效果。

## 从基础到专用架构

全连接网络是最通用的架构，但也是最低效的。

对于一张 224×224×3 的图片（约15万像素），
如果用全连接层直接处理，第一层的权重矩阵就有 15万×神经元数 个参数——
即使只有1000个神经元也是1.5亿个参数。
这太多了。

问题在于：全连接网络不利用数据的**结构信息**。

- 图片有空间结构——相邻像素之间的关系比远距离像素更重要
- 文本有序列结构——词的顺序携带了关键信息
- 不同位置的模式可能是相同的——图片左上角的"猫耳朵"和右下角的"猫耳朵"应该被同一个特征检测器识别

为了利用这些结构信息，出现了专用的网络架构：
- **CNN**：利用图片的空间局部性
- **RNN**：利用文本/时间序列的序列性
- **Transformer**：利用注意力机制直接建模任意位置之间的关系

**接下来三个模块就是这三个架构。**

## 关键认知

1. 神经元 = 加权求和 + 偏置 + 激活函数——极其简单的计算单元
2. 深度的意义在于层级抽象——每一层在前一层基础上构建更高级的特征
3. 激活函数引入非线性——没有它，再多层也等于一层
4. ReLU解决了梯度消失——正区间导数恒为1，梯度能顺利回传
5. 前向传播计算预测，反向传播计算梯度，梯度下降更新参数——训练的三步曲
6. 正则化（Dropout、权重衰减、批归一化）防止过拟合
7. 全连接网络不利用数据结构——专用架构（CNN、RNN、Transformer）应运而生
