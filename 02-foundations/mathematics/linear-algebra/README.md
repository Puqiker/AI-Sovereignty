# 线性代数 → AI

> 你学过向量、矩阵、矩阵乘法、特征值。
> 这个模块告诉你：这些概念在AI中不是背景知识，而是主角。
> 神经网络的每一层运算、词向量的语义空间、注意力机制的计算——全是线性代数。

---

## 向量：从数学对象到意义的容器

### 你已经知道的

向量是一个有序的数字列表。一个 n 维向量就是 n 个数字排在一起：

v = [3.2, -1.5, 0.8, 2.1]

你知道向量可以加减，可以缩放，可以计算点积。

### 它在AI中变成了什么

在AI中，向量被用来**表示意义**。

当一个大语言模型处理"猫"这个词时，
它不是在处理"猫"这个字符，
而是在处理一个几百维甚至几千维的向量，比如：
"猫" → [0.23, -0.87, 1.45, 0.02, ..., -0.56] # 768个数字


这个向量叫做**词嵌入（word embedding）**。

关键在于：这些数字不是随意的。
它们是模型在训练过程中学到的，
使得**意义相近的词在向量空间中的距离也相近**。

比如：
- "猫"和"狗"的向量之间的距离，小于"猫"和"汽车"的距离
- "国王"和"女王"的向量之间的关系，近似于"男人"和"女人"的关系

你学过的**向量点积**，在这里变成了**衡量两个概念相似度**的工具：

similarity(A, B) = A · B / (|A| × |B|)


这就是余弦相似度。你在线性代数里学过，
两个向量夹角越小，点积（归一化后）越大。
在AI中，这变成了：两个概念越相似，它们的向量夹角越小。

### 为什么这很重要

当你后面学到注意力机制时，你会发现它的核心计算就是：
用一个向量（查询）去和一组向量（键）计算点积，
找出哪些向量和当前查询最"相关"。

**注意力机制的本质，就是在高维向量空间中做相似度搜索。**

如果你没有"向量表示意义、点积衡量相似度"这个直觉，
注意力机制对你来说就是一堆看不懂的公式。
有了这个直觉，你一眼就能看透它在做什么。

## 矩阵乘法：从抽象运算到神经网络的核心

### 你已经知道的

矩阵乘法是这样的：一个 m×n 矩阵乘以一个 n×p 矩阵，得到一个 m×p 矩阵。

[a b] × [e f] [ae+bg af+bh]
[c d] [g h] = [ce+dg cf+dh]


你可能在学的时候觉得这东西很抽象，不知道它有什么用。

### 它在AI中变成了什么

**神经网络的每一层做的事情，本质上就是一次矩阵乘法加上一个非线性变换。**

具体来说，一个神经网络层的计算是：
输出 = 激活函数(输入 × 权重矩阵 + 偏置)


用数学符号写就是：

y = f(Wx + b)


其中：
- `x` 是输入向量（比如一个词的嵌入向量）
- `W` 是权重矩阵（这就是模型的"参数"，训练过程就是在调整这个矩阵里的数字）
- `b` 是偏置向量
- `f` 是激活函数（引入非线性，比如ReLU）
- `y` 是输出向量

**整个神经网络就是这个操作的反复叠加。**

第一层：`y₁ = f(W₁x + b₁)`
第二层：`y₂ = f(W₂y₁ + b₂)`
第三层：`y₃ = f(W₃y₂ + b₃)`
……

当人们说"一个模型有几十亿参数"时，
说的就是这些权重矩阵 W₁、W₂、W₃……里面的数字加起来有几十亿个。

训练模型的过程，就是调整这几十亿个数字，
使得模型在给定输入时能输出期望的结果。

### 为什么这很重要

当你理解了"每一层就是一次矩阵乘法"，很多事情就清晰了：

1. **为什么AI需要GPU**：GPU是专门为矩阵运算设计的硬件。CPU擅长复杂逻辑但矩阵运算慢，GPU擅长大规模并行矩阵运算。神经网络的计算几乎全是矩阵乘法，所以GPU是AI的标准硬件。

2. **为什么模型越大越贵**：参数越多，矩阵越大，每次推理需要的矩阵乘法运算量越大，消耗的算力和电力越多。

3. **为什么模型可以被压缩**：如果一个权重矩阵中很多数字接近于零，或者矩阵有低秩结构，就可以用更小的矩阵来近似它，这就是模型压缩的数学基础。

## 线性变换：从几何直觉到理解"层在做什么"

### 你已经知道的

矩阵乘法可以被理解为一种线性变换——
它把一个向量空间中的点映射到另一个向量空间中。

在二维空间里，一个 2×2 的矩阵可以做旋转、缩放、剪切、翻转。

### 它在AI中变成了什么

神经网络每一层的权重矩阵，就是在对数据做线性变换。

想象你的数据是高维空间中的一堆点。
每一层做的事情是：
1. 用矩阵乘法对这些点做线性变换（旋转、拉伸、投影）
2. 用激活函数做非线性扭曲（把空间"掰弯"）

通过反复的线性变换+非线性扭曲，
网络把原本纠缠在一起的数据点逐渐分离到不同的区域。

比如一个图像分类任务：
- 原始输入中，"猫"和"狗"的图片在像素空间中是混在一起的
- 经过多层变换后，在最后一层的输出空间中，"猫"的数据点都聚在一个区域，"狗"的数据点聚在另一个区域
- 分类只需要在这个最终空间中画一条分界线

**训练的目标就是找到合适的变换矩阵，使得数据在最终空间中可以被简单地分开。**

### 直觉

如果你曾经觉得线性代数里的"线性变换"很抽象——
现在你知道了，AI中的每一层神经网络，
就是在用一个具体的矩阵做一次具体的线性变换，
把你的数据从一种表示空间搬运到另一种表示空间。

## 特征值与特征向量：理解数据的主要方向

### 你已经知道的

一个方阵 A 的特征值 λ 和特征向量 v 满足：
Av = λv


特征向量是那些在矩阵变换下只发生缩放（不改变方向）的向量。
特征值是缩放的倍数。

### 它在AI中变成了什么

特征值和特征向量在AI中最直接的应用是**主成分分析（PCA）**——
一种降维方法。

高维数据（比如一张图片有几百万个像素）直接处理太昂贵了。
PCA的做法是：
1. 计算数据协方差矩阵的特征值和特征向量
2. 特征值最大的特征向量代表数据变化最大的方向（主成分）
3. 只保留前几个主成分，丢弃其余方向的信息

这样你用少量的维度就能保留数据的大部分信息。

在AI中，类似的思想出现在很多地方：
- 词嵌入的降维可视化（把几百维的词向量投影到二维来观察）
- 模型压缩中的低秩近似（一个大矩阵可以用几个小矩阵的乘积来近似，本质上就是保留主要特征方向）
- 理解训练动态（损失函数的Hessian矩阵的特征值分布能告诉你优化过程的难度）

### 为什么这很重要

你不需要在日常使用AI时计算特征值。
但当你看到"降维""低秩""主成分"这些概念时，
你的脑子里应该立刻浮现出特征值分解的画面——
找到数据中最重要的方向，忽略不重要的方向。

这个直觉会帮你理解后面遇到的很多技术决策的本质。

## 高维空间的直觉

有一个贯穿整个AI数学的概念需要你建立直觉：**高维空间**。

你的日常经验在三维空间中。
AI处理的向量动辄几百维、几千维。

高维空间有一些反直觉的性质：

1. **几乎所有的点都在"表面"附近**。在高维球体中，体积几乎全部集中在靠近表面的薄壳里。这意味着高维空间中的数据分布和低维空间非常不同。

2. **随机选择的两个向量几乎总是正交的**。在高维空间中，两个随机向量的余弦相似度趋近于零。这就是为什么高维嵌入空间能同时容纳大量不同的概念——每个概念可以指向空间中几乎不同的方向。

3. **距离的意义改变了**。在高维空间中，所有点到中心的距离趋于相同。这对聚类和最近邻搜索都有影响。

你不需要深入高维几何的数学细节。
但你需要知道：**你在二维、三维中建立的空间直觉，不能直接套用到高维中。**

当你看到AI论文中说"在嵌入空间中"做某种操作时，
你要意识到这个空间可能有几千维，
它的行为和你直觉中的三维空间不同。

## 关键映射总结

| 你学过的线性代数概念 | 在AI中的角色 |
|-------------------|-------------|
| 向量 | 表示意义（词嵌入、特征表示） |
| 向量点积 | 衡量相似度（注意力机制、检索） |
| 矩阵乘法 | 神经网络每一层的核心运算 |
| 矩阵（权重矩阵） | 模型的参数，训练就是在调整这些矩阵 |
| 线性变换 | 每一层把数据从一种表示空间变换到另一种 |
| 特征值/特征向量 | 找到数据的主要方向（降维、压缩、分析） |
| 高维向量空间 | AI操作的基本舞台 |
