# 数学基础

> 你学过的高等数学、线性代数、概率统计——这些不是和AI无关的旧知识。
> 它们就是AI的语言。
> 这个模块不重新教你数学。
> 它告诉你：你已经知道的那些数学，在AI里变成了什么。

---

## 前提

本模块假设你具备以下数学背景（大学本科理工科水平）：

- **线性代数**：向量、矩阵、矩阵乘法、转置、逆矩阵、特征值与特征向量
- **微积分**：导数、偏导数、链式法则、多元函数、梯度
- **概率统计**：概率基本概念、条件概率、贝叶斯公式、常见分布（正态、均匀等）、期望与方差

如果你对以上概念感到完全陌生，建议先通过大学数学教材补齐基础再进入本模块。

如果你"学过但忘了"，不用担心——每个模块会在需要时快速唤醒你的记忆，
然后直接桥接到AI中的应用。重点不在于你能不能做复杂的数学推导，
而在于你能不能建立**数学直觉**——看到一个AI概念时，
能在脑子里映射到对应的数学图景。

---

## 核心观点

**AI不是魔法。AI是数学。**

当你理解了AI背后的数学，你就彻底祛魅了——
你看到的不再是一个神秘的"智能"系统，
而是一个由你熟悉的数学运算堆叠起来的计算过程。

这种去魅不会让AI变得不那么有用。
它会让你变得更有判断力。

## 学习顺序

| 顺序 | 模块 | 核心桥梁 |
|------|------|---------|
| 1 | [线性代数 → AI](linear-algebra/) | 你学过的矩阵运算 = 神经网络的核心计算 |
| 2 | [概率统计 → AI](probability-statistics/) | 你学过的概率分布 = 模型输出的本质 |
| 3 | [微积分与优化 → AI](calculus-optimization/) | 你学过的链式法则 = 反向传播算法 |
| 4 | [信息论基础](information-theory/) | 你可能没学过，但它是AI训练的度量语言 |

## 完成标准

当你完成数学基础模块后，你应该能够：

- [ ] 看到"神经网络的一层"时，脑子里浮现的是矩阵乘法 + 非线性变换
- [ ] 理解为什么AI的输出是概率分布，而不是确定性答案
- [ ] 解释反向传播的本质就是链式法则在计算图上的应用
- [ ] 知道交叉熵损失函数在度量什么，为什么它是这样定义的
- [ ] 面对AI论文中的数学公式时，不再感到恐惧，而是能识别出熟悉的数学结构

完成后，继续进入 [机器学习核心](../machine-learning/)。
