# AI基础设施

> 前面的模块告诉你AI在算法和数学层面是怎么工作的。
> 这个模块告诉你：**支撑这一切的物理和工程基础是什么。**
> 为什么AI需要GPU而不是CPU？几千亿参数的模型怎么训练？
> 模型训练好之后怎么部署给用户用？数据是怎么准备和流动的？
>
> 你不需要成为基础设施工程师，
> 但你需要理解这些概念，因为它们直接影响：
> - 你能用什么规模的模型
> - 你的推理成本是多少
> - 你的私有部署方案应该怎么设计
> - 你对AI系统性能瓶颈的判断力
