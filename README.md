# 🛡️ AI-Sovereignty

**From Understanding to Mastery — Reclaim Your Sovereignty in the Age of AI**

**从理解到精通——在AI时代夺回你的主权**

---

## 这个项目是什么

AI-Sovereignty 是一个开源的知识体系和学习平台，
帮助任何人——从完全零基础到高级从业者——建立对AI的完整认知，
安全地使用AI，最终具备驾驭AI的能力。

它不是一个工具推荐清单。
它不是一个速成教程。
它是一条从觉醒到主权的完整路径。

> 📜 阅读 [项目宣言（MANIFESTO）](MANIFESTO.md) 了解我们的信念。

---

## 为什么需要这个项目

AI工具层出不穷。大多数人的使用方式停留在功能层面——
能对话、能生成图片、能写代码，按照官方的安全指南做好配置，
然后就不再深入了。

这条线就是一道分水岭。

**线以下**：你使用AI，但你不理解它。
你不知道它为什么会幻觉，不知道你的数据去了哪里，
不知道为什么它有时很聪明有时很蠢。
你按别人说的做安全配置，但你不知道为什么。

**线以上**：你理解AI的本质——概率模型、神经网络、注意力机制。
你知道任何新工具的底层实现思路不会发生质变。
你能自己判断一个AI系统是否安全，能为企业建立安全体系。
你不追逐工具，因为你掌握了穿越工具迭代的能力。

这个项目的存在，就是为了把你拉到线以上。

## 学习路径

### 🧠 [01 觉醒（Awakening）](01-awakening/)
认知启蒙。理解AI的本质与人的独特性。建立独立思考的纪律。

- AI到底是什么 / 人类心智 vs AI / 为什么必须理解底层原理
- 认知卫生：AI依赖陷阱与独立思考纪律
- 用AI学AI的悖论：如何验证AI给你的知识

### 🔬 [02 筑基（Foundations）](02-foundations/)
底层原理。数学基础、机器学习、深度学习、大模型内核、数据、评估、Prompt原理。

- 数学直觉：线性代数 / 概率统计 / 微积分与优化 / 信息论
- 机器学习核心：监督 / 无监督 / 强化学习
- 深度学习：神经网络 / CNN / RNN / 注意力机制 / Transformer
- 大模型内核：分词 / 预训练 / 微调 / RLHF / 推理机制 / 幻觉分析
- 数据基础：数据质量 / 偏差检测 / 数据集构建
- 模型评估：超越benchmark / 构建自己的测试集 / 失败模式分析
- 基于原理的Prompt Engineering

### ⚔️ [03 安全（Security）](03-security/)
攻防兼备的AI安全体系。从威胁理解到体系建设。

- 威胁全景：Prompt注入 / 数据投毒 / 模型窃取 / 对抗攻击 / 隐私泄露
- 供应链安全：模型权重篡改 / 数据集投毒 / 恶意插件
- 主流AI工具安全使用指南
- AI安全体系建设：风险评估 / 防御架构 / 红队方法 / 应急响应
- 法律与合规：全球AI监管全景 / 行业特定合规
- 漏洞动态追踪 / 实战安全实验室

### 🚀 [04 精通（Mastery）](04-mastery/)
从使用者到创造者。构建AI系统的完整能力。

- 进阶技术：多模态 / Agent架构 / RAG / AI推理 / 模型可解释性
- 构建AI解决方案：场景发现方法论 / 工作流编排 / AI调试 / AI测试
- 创造AI工具
- AI系统运维：性能监控 / 数据漂移 / 成本优化
- 避免供应商锁定：抽象层设计 / 多供应商架构 / 迁移策略

### 🌌 [05 超越（Transcendence）](05-transcendence/)
协同性进步。在认知维度上领先于AI。

- 认知进化：AI时代的心智模型 / 创造力放大 / 潜意识与直觉
- 持续认知卫生：深度工作 / 刻意AI断食 / 强化AI做不到的能力
- 人机协同方法论：AI增强的工作流 / 经济主权 / 一个人的公司
- 前沿认知：个体 vs 集体 / AI哲学 / 未来场景
