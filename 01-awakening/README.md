# 🧠 01 觉醒（Awakening）

> 你走进来时，可能以为AI就是聊天机器人。
> 你走出去时，你将理解AI的本质边界，也理解人的独特性。
> 你将建立独立思考的纪律，知道在AI时代如何保持清醒。

---

## 为什么从这里开始

在你学习任何技术细节之前，你需要先回答几个问题：

- AI到底是什么？它和你以为的那个东西一样吗？
- 人的思维和AI的"思维"，区别到底在哪里？
- 为什么理解底层原理如此重要，而不是直接学怎么用？
- 在AI时代，你的认知需要发生什么样的转变？
- 当AI越来越能帮你想的时候，你自己的思考能力怎么办？
- 当你用AI来学习AI知识时，怎么知道它教你的是对的？

这六个问题对应这个阶段的六个模块。
它们不是技术问题，而是认知问题。
认知不对，后面学的所有技术都建立在错误的地基上。

## 完成标准

当你完成这个阶段时，你应该能够：

- [ ] 用自己的话解释AI的工作本质，而不是用"智能""理解""思考"这类拟人化的词
- [ ] 说出至少三个人类心智具备而AI不具备的核心能力，并解释为什么
- [ ] 解释为什么两个人用同一个AI工具，一个人能穿透它的行为，另一个人只能跟着它走
- [ ] 识别出自己在日常中对AI产生认知依赖的具体场景
- [ ] 当AI给你一个看似正确的解释时，知道至少两种方法来验证它

完成后，进入 [02 筑基（Foundations）](../02-foundations/)。
