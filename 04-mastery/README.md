# 🚀 04 精通（Mastery）

> 你走进来时，你会用AI工具。
> 你走出去时，你能创造AI工具和全新的AI应用场景。
> 你能编排复杂的AI工作流，能调试AI系统的异常行为，
> 能为概率性系统设计测试，能运维生产环境中的AI。
> 你从AI的消费者变成了AI的生产者。

---

## 这个阶段和前面的区别

前三个阶段你学到了"AI是什么""AI怎么工作""AI有什么风险"。
这个阶段你要回答的问题是："**我怎么用AI解决真实世界的问题。**"

这不是简单地调用API——而是：
- 面对一个业务问题，判断AI是否是正确的解决方案
- 如果是，设计一个完整的AI系统架构
- 构建、测试、部署、运维这个系统
- 确保它在生产环境中可靠、安全、经济

## 完成标准

当你完成精通阶段时，你应该能够：

- [ ] 面对一个业务问题，判断是否适合用AI解决，以及用哪种AI技术
- [ ] 设计一个包含多个AI组件的系统架构
- [ ] 为AI系统设计测试策略，包括概率性输出的评估方案
- [ ] 调试AI系统的异常行为——定位问题是在输入、模型、还是系统层面
- [ ] 运维生产环境中的AI系统——监控性能、管理成本、处理漂移
- [ ] 设计多供应商架构，避免被单一AI厂商绑定

完成后，进入 [05 超越（Transcendence）](../05-transcendence/)。
