# 开源贡献

---

## 为什么参与AI开源

- 加速你的学习——阅读和贡献优质开源项目是最好的学习方式
- 建立影响力——在AI社区中建立你的专业声誉
- 回馈社区——你从开源中获益，也应该贡献回去

## 参与方式

### 入门级
- 提交bug报告
- 改进文档
- 翻译文档

### 进阶
- 修复bug
- 实现小的新功能
- 编写测试

### 高级
- 设计和实现新功能
- 参与架构讨论
- 维护项目的某个模块

## 推荐关注的AI开源项目

- 推理引擎：vLLM、llama.cpp、Ollama
- 编排框架：LangChain、LlamaIndex
- 模型：HuggingFace Transformers
- 向量数据库：Chroma、FAISS
- 评估工具：lm-evaluation-harness
